Remarks

               The bag-of-words (BoW) model can be used to extract finite
               length features from otherwise varying length representations.
               This class can uses any feature
               detector to determine a coded representation for a given image.

               This is the most generic version for the BoW model, which can accept any
               choice of IFeatureDetector TPoint  for any kind of point,
               even non-numeric ones. This class can also support any clustering algorithm
               as well. 
Examples

               In this example, we will create a Bag-of-Words to operate on byte[] vectors,
               which otherwise wouldn't be supported by the simpler BoW version. Those byte vectors
               are composed of binary features detected by a FastRetinaKeypointDetector.
               In order to cluster those features, we will be using a KModes TData 
               algorithm with a matching template argument to make all constructors happy:
