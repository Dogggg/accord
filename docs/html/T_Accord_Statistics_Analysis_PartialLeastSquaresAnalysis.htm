Remarks

Partial least squares regression (PLS-regression) is a statistical method that bears some relation to principal components regression; instead of finding hyperplanes of maximum variance between the response and independent variables, it finds a linear regression model by projecting the predicted variables and the observable variables to a new space. Because both the X and Y data are projected to new spaces, the PLS family of methods are known as bilinear factor models.

Examples

CopyC#
 1// Following the small example by Hervé Abdi (Hervé Abdi, Partial Least Square Regression),
 2// we will create a simple example where the goal is to predict the subjective evaluation of
 3// a set of 5 wines. The dependent variables that we want to predict for each wine are its 
 4// likeability, and how well it goes with meat, or dessert (as rated by a panel of experts).
 5// The predictors are the price, the sugar, alcohol, and acidity content of each wine.
 6
 7
 8// Here we will list the inputs, or characteristics we would like to use in order to infer
 9// information from our wines. Each row denotes a different wine and lists its corresponding
10// observable characteristics. The inputs are usually denoted by X in the PLS literature.
11
12double[,] inputs = 
13{
14    // Wine | Price | Sugar | Alcohol | Acidity
15    {           7,     7,      13,        7     },
16    {           4,     3,      14,        7     },
17    {          10,     5,      12,        5     },
18    {          16,     7,      11,        3     },
19    {          13,     3,      10,        3     },
20 };
21
22
23 // Here we will list our dependent variables. Dependent variables are the outputs, or what we
24 // would like to infer or predict from our available data, given a new observation. The outputs
25 // are usually denoted as Y in the PLS literature.
26
27 double[,] outputs = 
28 {
29     // Wine | Hedonic | Goes with meat | Goes with dessert
30     {           14,          7,                 8         },
31     {           10,          7,                 6         },
32     {            8,          5,                 5         },
33     {            2,          4,                 7         },
34     {            6,          2,                 4         },
35 };
36
37
38 // Next, we will create our Partial Least Squares Analysis passing the inputs (values for 
39 // predictor variables) and the associated outputs (values for dependent variables).
40
41 // We will also be using the using the Covariance Matrix/Center method (data will only
42 // be mean centered but not normalized) and the SIMPLS algorithm. 
43 PartialLeastSquaresAnalysis pls = new PartialLeastSquaresAnalysis(inputs, outputs,
44     AnalysisMethod.Center, PartialLeastSquaresAlgorithm.SIMPLS);
45
46 // Compute the analysis with all factors. The number of factors
47 // could also have been specified in a overload of this method.
48
49 pls.Compute();
50
51 // After computing the analysis, we can create a linear regression model in order
52 // to predict new variables. To do that, we may call the CreateRegression() method.
53
54 MultivariateLinearRegression regression = pls.CreateRegression();
55
56 // After the regression has been created, we will be able to classify new instances. 
57 // For example, we will compute the outputs for the first input sample:
58
59 double[] y = regression.Compute(new double[] { 7, 7, 13, 7 });
60
61 // The y output will be very close to the corresponding output used as reference.
62 // In this case, y is a vector of length 3 with values { 13.98, 7.00, 7.75 }.
