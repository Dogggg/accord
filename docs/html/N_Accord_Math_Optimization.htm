Accord.Math.Optimization Namespace Accord.NET Framework
Show table of contents (goes to the online documentation index).
Contains classes for constrained and unconstrained optimizaion. Includes Conjugate Gradient (CG), Broyden–Fletcher–Goldfarb–Shanno (BFGS), and the Goldfarb-Idnani solver for Quadratic Programming (QP) problems.
