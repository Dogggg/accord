The SupportVectorMachine type exposes the following members.
Properties
  NameDescription
Inputs
Gets the number of inputs accepted by this machine.
IsCompact
Gets whether this machine is in compact mode. Compact machines do not need to keep storing their support vectors.
IsProbabilistic
Gets a value indicating whether this machine produces probabilistic outputs.
Link
Gets or sets the link function used by this machine, if any.
SupportVectors
Gets or sets the collection of support vectors used by this machine.
Threshold
Gets or sets the threshold (bias) term for this machine.
Weights
Gets or sets the collection of weights used by this machine.
