Remarks

Without question, the most popular models available in this namespace are the Support Vector Machines. The inner VectorMachines namespace contains both standard SupportVectorMachines and their kernel extension given by KernelSupportVectorMachines. For multiple classes or categories, the framework also offers MulticlassSupportVectorMachines and MultilabelSupportVectorMachines. Multi-class machines can be used for cases where a single class should be picked up from a list of several class labels, and the multi-label machine for cases where multiple class labels might be detected for a single input vector. The multi-class machines also support two types of classification: the faster decision based on Decision Directed Acyclic Graphs, and the more traditional based on a Voting scheme.

Still talking about SVMs, the framework offers several methods to learn and train those machines. While the most popular method will always be the SequentialMinimalOptimization (SMO) algorithm, the framework can also learn Least Squares SVMs (LS-SVMs) using LeastSquaresLearning, and even calibrate SVMs to produce probabilistic outputs using ProbabilisticOutputLearning. A huge variety of kernels functions is available in the statistics namespace, and new kernels can be created easily using the IKernel interface.

Besides SVMs, this namespace includes Naïve Bayes, Decision Trees, Boosting methods, Clustering techniques, and other diverse classifiers, such as the KNearestNeighbors, and the much simpler minimum (mean) distance classifier.

To measure the performance of all those models, it is possible to choose from several performance assessment techniques such as Bootstrap, k-Fold Cross-Validation (CV) and split-set validation.
