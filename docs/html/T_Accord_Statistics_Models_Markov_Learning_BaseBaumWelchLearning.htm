Remarks

This class uses a template method pattern so specialized classes can be written for each kind of hidden Markov model emission density (either discrete or continuous). The methods UpdateEmissions()()()(), ComputeForwardBackward(Int32, array<Double,2>[,](,)[,][,], array<Double,2>[,](,)[,][,]) and ComputeKsi(Int32, array<Double,2>[,](,)[,][,], array<Double,2>[,](,)[,][,]) should be overriden by inheriting classes to specify how those probabilities should be computed for the density being modeled.
