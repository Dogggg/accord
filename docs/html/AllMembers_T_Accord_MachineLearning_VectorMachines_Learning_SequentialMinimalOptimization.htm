Properties

ActiveExamples
Gets the indices of the active examples (examples which have the corresponding Lagrange multiplier different than zero).
BoundedExamples
Gets the indices of the examples at the boundary (examples which have the corresponding Lagrange multipliers equal to C).
CacheSize
Gets or sets the cache size to partially stored the kernel matrix. Default is the same number of input vectors.
Compact
Gets or sets whether to produce compact models. Compact formulation is currently limited to linear models.
Complexity
Complexity (cost) parameter C. Increasing the value of C forces the creation of a more accurate model that may not generalize well. Default value is the number of examples divided by the trace of the kernel matrix.
Epsilon
Epsilon for round-off errors. Default value is 1e-12.
Lagrange
Gets the value for the Lagrange multipliers (alpha) for every observation vector.
NegativeWeight
Gets or sets the negative class weight. This should be a value between 0 and 1 indicating how much of the Complexity parameter C should be applied to instances carrying the negative label.
NonBoundExamples
Gets the indices of the non-bounded examples (examples which have the corresponding Lagrange multipliers between 0 and C).
PositiveWeight
Gets or sets the positive class weight. This should be a value between 0 and 1 indicating how much of the Complexity parameter C should be applied to instances carrying the positive label.
Strategy
Gets or sets the pair selection streategy to be used during optimization.
Tolerance
Convergence tolerance. Default value is 1e-2.
UseClassProportions
Gets or sets a value indicating whether the weight ratio to be used between Complexity values for negative and positive instances should be computed automatically from the data proportions. Default is false.
UseComplexityHeuristic
Gets or sets a value indicating whether the Complexity parameter C should be computed automatically by employing an heuristic rule. Default is false.
WeightRatio
Gets or sets the weight ratio between positive and negative class weights. This ratio controls how much of the Complexity parameter C should be applied to the positive class.
