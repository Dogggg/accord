Accord.Statistics.Analysis Namespace Accord.NET Framework

Contains many statistical analysis, such as PCA, LDA, KPCA, KDA, PLS, IDA, Logistic Regression and Stepwise Logistic Regression Analyses. Also contains performance assessment analysis such as contingency tables and ROC curves.
