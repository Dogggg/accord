Remarks

In statistics, logistic regression (sometimes called the logistic model or logit model) is used for prediction of the probability of occurrence of an event by fitting data to a logistic curve. It is a generalized linear model used for binomial regression.

Like many forms of regression analysis, it makes use of several predictor variables that may be either numerical or categorical. For example, the probability that a person has a heart attack within a specified time period might be predicted from knowledge of the person's age, sex and body mass index.

Logistic regression is used extensively in the medical and social sciences as well as marketing applications such as prediction of a customer's propensity to purchase a product or cease a subscription.

 1// Suppose we have the following data about some patients.
 2// The first variable is continuous and represent patient
 3// age. The second variable is dicotomic and give whether
 4// they smoke or not (This is completely fictional data).
 5double[][] input =
 6{
 7    new double[] { 55, 0 }, // 0 - no cancer
 8    new double[] { 28, 0 }, // 0
 9    new double[] { 65, 1 }, // 0
10    new double[] { 46, 0 }, // 1 - have cancer
11    new double[] { 86, 1 }, // 1
12    new double[] { 56, 1 }, // 1
13    new double[] { 85, 0 }, // 0
14    new double[] { 33, 0 }, // 0
15    new double[] { 21, 1 }, // 0
16    new double[] { 42, 1 }, // 1
17};
18
19// We also know if they have had lung cancer or not, and 
20// we would like to know whether smoking has any connection
21// with lung cancer (This is completely fictional data).
22double[] output =
23{
24    0, 0, 0, 1, 1, 1, 0, 0, 0, 1
25};
26
27
28// To verify this hypothesis, we are going to create a logistic
29// regression model for those two inputs (age and smoking).
30LogisticRegression regression = new LogisticRegression(inputs: 2);
31
32// Next, we are going to estimate this model. For this, we
33// will use the Iteravely reweighted least squares method.
34var teacher = new IterativeReweightedLeastSquares(regression);
35
36// Now, we will iteratively estimate our model. The Run method returns
37// the maximum relative change in the model parameters and we will use
38// it as the convergence criteria.
39
40double delta = 0;
41do
42{
43    // Perform an iteration
44    delta = teacher.Run(input, output);
45
46} while (delta > 0.001);
47
48// At this point, we can compute the odds ratio of our variables.
49// In the model, the variable at 0 is always the intercept term, 
50// with the other following in the sequence. Index 1 is the age
51// and index 2 is whether the patient smokes or not.
52
53// For the age variable, we have that individuals with
54//   higher age have 1.021 greater odds of getting lung
55//   cancer controlling for cigarrete smoking.
56double ageOdds = regression.GetOddsRatio(1); // 1.0208597028836701
57
58// For the smoking/non smoking category variable, however, we
59//   have that individuals who smoke have 5.858 greater odds
60//   of developing lung cancer compared to those who do not 
61//   smoke, controlling for age (remember, this is completely
62//   fictional and for demonstration purposes only).
63double smokeOdds = regression.GetOddsRatio(2); // 5.8584748789881331
