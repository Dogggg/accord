Remarks

Multivariate Linear Regression is a generalization of Multiple Linear Regression to allow for multiple outputs.

Examples

CopyC#
 1// The multivariate linear regression is a generalization of
 2// the multiple linear regression. In the multivariate linear
 3// regression, not only the input variables are multivariate,
 4// but also are the output dependent variables.
 5
 6// In the following example, we will perform a regression of
 7// a 2-dimensional output variable over a 3-dimensional input
 8// variable.
 9
10double[][] inputs = 
11{
12    // variables:  x1  x2  x3
13    new double[] {  1,  1,  1 }, // input sample 1
14    new double[] {  2,  1,  1 }, // input sample 2
15    new double[] {  3,  1,  1 }, // input sample 3
16};
17
18double[][] outputs = 
19{
20    // variables:  y1  y2
21    new double[] {  2,  3 }, // corresponding output to sample 1
22    new double[] {  4,  6 }, // corresponding output to sample 2
23    new double[] {  6,  9 }, // corresponding output to sample 3
24};
25
26// With a quick eye inspection, it is possible to see that
27// the first output variable y1 is always the double of the
28// first input variable. The second output variable y2 is
29// always the triple of the first input variable. The other
30// input variables are unused. Nevertheless, we will fit a
31// multivariate regression model and confirm the validity
32// of our impressions:
33
34// Create a new multivariate linear regression with 3 inputs and 2 outputs
35var regression = new MultivariateLinearRegression(3, 2);
36
37// Now, compute the multivariate linear regression:
38double error = regression.Regress(inputs, outputs);
39
40// At this point, the regression error will be 0 (the fit was
41// perfect). The regression coefficients for the first input
42// and first output variables will be 2. The coefficient for
43// the first input and second output variables will be 3. All
44// others will be 0.
45// 
46// regression.Coefficients should be the matrix given by
47// 
48// double[,] coefficients = {
49//                              { 2, 3 },
50//                              { 0, 0 },
51//                              { 0, 0 },
52//                          };
53// 
54
55// The first input variable coefficients will be 2 and 3:
56Assert.AreEqual(2, regression.Coefficients[0, 0], 1e-10);
57Assert.AreEqual(3, regression.Coefficients[0, 1], 1e-10);
58
59// And all other coefficients will be 0:
60Assert.AreEqual(0, regression.Coefficients[1, 0], 1e-10);
61Assert.AreEqual(0, regression.Coefficients[1, 1], 1e-10);
62Assert.AreEqual(0, regression.Coefficients[2, 0], 1e-10);
63Assert.AreEqual(0, regression.Coefficients[2, 1], 1e-10);
64
65// We can also check the r-squared coefficients of determination:
66double[] r2 = regression.CoefficientOfDetermination(inputs, outputs);
67
68// Which should be one for both output variables:
69Assert.AreEqual(1, r2[0]);
70Assert.AreEqual(1, r2[1]);
