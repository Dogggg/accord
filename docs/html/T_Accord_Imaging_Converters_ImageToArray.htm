Remarks

This class converts images to single or jagged arrays of either double-precision or single-precision floating-point values.

Examples

This example converts a 16x16 Bitmap image into a double[] array with values between 0 and 1.

CopyC#
 1// Obtain a 16x16 bitmap image
 2// Bitmap image = ...
 3
 4// Show on screen
 5ImageBox.Show(image, PictureBoxSizeMode.Zoom);
 6
 7// Create the converter to convert the image to an
 8//   array containing only values between 0 and 1 
 9ImageToArray conv = new ImageToArray(min: 0, max: 1);
10
11// Convert the image and store it in the array
12double[] array; conv.Convert(image, out array);
13
14// Show the array on screen
15ImageBox.Show(array, 16, 16, PictureBoxSizeMode.Zoom);    ///
